КАК ОТКРЫТЬ ДИАГРАММЫ
=====================

Вариант 1 (рекомендуется) — VS Code:
  1. Установить расширение "PlantUML" (jebbs.plantuml)
  2. Открыть любой .puml файл
  3. Alt+D — предпросмотр диаграммы

Вариант 2 — онлайн:
  1. Открыть https://www.plantuml.com/plantuml/uml/
  2. Вставить содержимое .puml файла
  3. Скриншот готовой диаграммы

ФАЙЛЫ
=====
usecase.puml          — диаграмма вариантов использования (Use Case)
er.puml               — ER-диаграмма базы данных
components.puml       — диаграмма компонентов системы
sequence_checkout.puml — последовательность: оформление заказа
sequence_ws.puml      — последовательность: WebSocket уведомления
sequence_jwt.puml     — последовательность: JWT авторизация
